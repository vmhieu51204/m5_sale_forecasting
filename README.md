# M5 Sales Forecasting

This project implements a sales forecasting pipeline for the M5 Forecasting - Accuracy competition dataset. It utilizes a variety of models including LightGBM, LSTM (PyTorch), and statistical models (ARIMA, SARIMAX, Exponential Smoothing) to predict future sales.

## Dataset
Daily sales data of Walmart recorded from 2011-01-29 to 2016-06-19. Each item is categorized into categories and
departments and is identified by a unique ”item id”

Link: https://www.kaggle.com/c/m5-forecasting-accuracy/data
 
## Requirements

The project requires the following Python libraries:

- `numpy`
- `pandas`
- `lightgbm`
- `torch`
- `statsmodels`
- `scikit-learn`
- `tqdm`
